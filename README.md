https://akozyreva.github.io/nuxt-gh-pages/

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project and deploy it to gh-pagees branch
$ npm run generate
$ npm run deploy
```
